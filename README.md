# 🧠 Brain Tumor MRI Classifier (PyTorch + Docker + Streamlit + Cloud Deployment)

A deep learning web application that classifies brain tumor MRI scans into 4 categories using a PyTorch CNN.  
Deployed as a Docker container and accessible via browser with real-time prediction.

---

### 🔥 Features
- 4-class tumor prediction using PyTorch CNN  
- Streamlit UI for image upload + confidence output  
- Docker-containerized deployment  
- Model inference runs fully in cloud (no local setup needed)

---

### 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
