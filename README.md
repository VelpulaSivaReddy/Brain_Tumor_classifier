# 🧠 Brain Tumor Classifier

A Deep Learning-based web application that classifies brain MRI images into different tumor categories using Convolutional Neural Networks (CNN). This project helps in early detection of brain tumors with fast and accurate predictions.

---

## 🌐 Live Demo

👉 https://brain-tumor-app-cq66.onrender.com

---

## 🚀 Features

- Upload MRI scan images
- Automatic tumor classification
- Supports multiple tumor types
- Clean and user-friendly interface
- Real-time predictions
- High accuracy deep learning model

---

## 🧩 Problem Statement

Brain tumors are critical medical conditions that require early diagnosis for effective treatment. Manual analysis of MRI scans can be time-consuming and error-prone. This project leverages Artificial Intelligence to automate tumor classification, improving speed and reliability.

## 🧪 Model Details

- **Model Type:** CNN (Convolutional Neural Network)
- **Input:** Brain MRI images
- **Output Classes:**
  - Glioma Tumor
  - Meningioma Tumor
  - Pituitary Tumor
  - No Tumor
